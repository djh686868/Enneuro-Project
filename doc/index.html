EnNeuro

轻量级深度学习框架，专注于卷积神经网络的设计与实现

概述

EnNeuro 是一个轻量级的深度学习框架，专为卷积神经网络设计。它提供了简洁的API接口，支持张量操作、自动微分、神经网络模块、数据加载和模型训练等功能。

主要特点

简洁的API

提供类似PyTorch的简洁API，易于学习和使用

自动微分

支持自动微分，简化模型训练过程

模块化设计

采用模块化设计，便于扩展和定制

支持GPU加速

支持GPU加速，提高训练效率

安装指南

EnNeuro 框架的安装非常简单，只需克隆代码库并确保依赖项正确安装。

环境要求

  • Python 3.8+
  • NumPy
  • OpenCV (用于可视化)

安装步骤

# 克隆代码库 git clone https://github.com/yourusername/Enneuro-Project.git # 进入项目目录 cd Enneuro-Project # 安装依赖 pip install numpy opencv-python

快速开始

以下是一个简单的示例，展示如何使用 EnNeuro 框架创建和训练一个简单的神经网络。

基本示例

import numpy as np from eneuro.base import Tensor, linear, relu, softmax from eneuro.nn import Module, MSELoss, SGD # 创建一个简单的神经网络 class SimpleNet(Module): def __init__(self): super().__init__() self.fc1 = linear(2, 10) self.fc2 = linear(10, 1) def forward(self, x): x = relu(self.fc1(x)) x = self.fc2(x) return x # 创建模型、损失函数和优化器 model = SimpleNet() criterion = MSELoss() optimizer = SGD(model.parameters(), lr=0.01) # 准备数据 X = Tensor(np.random.randn(100, 2)) y = Tensor(np.random.randn(100, 1)) # 训练模型 for epoch in range(100): # 前向传播 output = model(X) loss = criterion(output, y) # 反向传播 optimizer.zero_grad() loss.backward() optimizer.step() if (epoch + 1) % 10 == 0: print(f'Epoch [{epoch+1}/100], Loss: {loss.data:.4f}')

核心功能

张量操作

EnNeuro 的核心是 Tensor 类，它提供了丰富的张量操作功能。

from eneuro.base import Tensor # 创建张量 x = Tensor(np.array([1, 2, 3])) y = Tensor(np.array([4, 5, 6])) # 基本运算 z = x + y print(z) # 自动微分 x = Tensor(np.array([1.0]), requires_grad=True) y = x ** 2 y.backward() print(x.grad)

神经网络模块

EnNeuro 提供了丰富的神经网络模块，包括线性层、卷积层、激活函数等。

from eneuro.base import conv2d, relu, pooling from eneuro.nn import Module class CNN(Module): def __init__(self): super().__init__() # 卷积层 self.conv1 = conv2d(1, 16, kernel_size=3, stride=1, padding=1) self.conv2 = conv2d(16, 32, kernel_size=3, stride=1, padding=1) def forward(self, x): x = relu(self.conv1(x)) x = pooling(x, kernel_size=2, stride=2) x = relu(self.conv2(x)) x = pooling(x, kernel_size=2, stride=2) return x

数据加载

EnNeuro 提供了数据加载和批处理功能，方便处理大型数据集。

from eneuro.data import Dataset, DataLoader # 自定义数据集 class CustomDataset(Dataset): def __init__(self, data, labels): self.data = data self.labels = labels def __len__(self): return len(self.data) def __getitem__(self, idx): return self.data[idx], self.labels[idx] # 创建数据加载器 dataset = CustomDataset(X, y) dataloader = DataLoader(dataset, batch_size=32, shuffle=True) # 遍历数据 for batch_data, batch_labels in dataloader: # 处理批次数据 pass

API 参考

base 模块

base 模块包含核心的张量操作和基础函数。

Tensor

张量类，支持自动微分和各种张量操作。

参数
  • data - 张量数据，可以是NumPy数组
  • requires_grad - 是否需要计算梯度，默认为False
  • name - 张量名称，可选
方法
  • backward() - 执行反向传播，计算梯度
  • reshape(shape) - 重塑张量形状
  • transpose(axes) - 转置张量
  • argmax(axis) - 返回指定轴上的最大值索引
  • sum(axis, keepdims) - 对指定轴求和
示例
# 创建张量 x = Tensor(np.array([1, 2, 3])) # 自动微分 x = Tensor(np.array([1.0]), requires_grad=True) y = x ** 2 y.backward() print(x.grad) # 输出: Tensor([2.])

linear

线性层，实现全连接操作。

参数
  • in_features - 输入特征维度
  • out_features - 输出特征维度
  • bias - 是否使用偏置，默认为True
返回值

线性变换后的张量

示例
# 创建线性层 fc = linear(10, 5) # 使用线性层 x = Tensor(np.random.randn(32, 10)) y = fc(x) print(y.shape) # 输出: (32, 5)

conv2d

二维卷积层，用于图像处理。

参数
  • in_channels - 输入通道数
  • out_channels - 输出通道数
  • kernel_size - 卷积核大小
  • stride - 步长，默认为1
  • padding - 填充，默认为0
返回值

卷积后的张量

示例
# 创建卷积层 conv = conv2d(3, 16, kernel_size=3, stride=1, padding=1) # 使用卷积层 x = Tensor(np.random.randn(32, 3, 64, 64)) y = conv(x) print(y.shape) # 输出: (32, 16, 64, 64)

relu

ReLU激活函数。

参数
  • x - 输入张量
返回值

应用ReLU后的张量

示例
x = Tensor(np.array([-1, 0, 1])) y = relu(x) print(y) # 输出: Tensor([0, 0, 1])

nn 模块

nn 模块包含神经网络相关的类和函数。

Module

神经网络模块基类，所有网络层和模型都继承自此类。

方法
  • forward(x) - 前向传播方法，需要子类实现
  • parameters() - 返回模型的所有参数
  • zero_grad() - 清空所有参数的梯度
  • load_state_dict(state_dict) - 加载模型状态
  • state_dict() - 返回模型状态
示例
class SimpleNet(Module): def __init__(self): super().__init__() self.fc1 = linear(2, 10) self.fc2 = linear(10, 1) def forward(self, x): x = relu(self.fc1(x)) x = self.fc2(x) return x model = SimpleNet()

MSELoss

均方误差损失函数。

参数
  • pred - 预测值
  • target - 目标值
返回值

均方误差损失值

示例
criterion = MSELoss() pred = Tensor(np.array([1.0, 2.0])) target = Tensor(np.array([1.5, 2.5])) loss = criterion(pred, target) print(loss) # 输出: Tensor([0.25])

SGD

随机梯度下降优化器。

参数
  • parameters - 模型参数列表
  • lr - 学习率
  • momentum - 动量，默认为0
  • weight_decay - 权重衰减，默认为0
方法
  • zero_grad() - 清空梯度
  • step() - 执行一步优化
示例
optimizer = SGD(model.parameters(), lr=0.01, momentum=0.9) optimizer.zero_grad() loss.backward() optimizer.step()

data 模块包含数据处理相关的类。

数据集基类，用于创建自定义数据集。

示例
class CustomDataset(Dataset): def __init__(self, data, labels): self.data = data self.labels = labels def __len__(self): return len(self.data) def __getitem__(self, idx): return self.data[idx], self.labels[idx]

DataLoader

数据加载器，用于批量加载数据。

参数
  • dataset - 数据集对象
  • batch_size - 批次大小
  • shuffle - 是否打乱数据，默认为False
  • num_workers - 工作线程数，默认为0
示例
dataloader = DataLoader(dataset, batch_size=32, shuffle=True) for batch_data, batch_labels in dataloader: # 处理批次数据 pass

train 模块

train 模块包含模型训练相关的类。

Trainer

模型训练器，用于训练模型。

参数
  • model - 模型对象
  • criterion - 损失函数
  • optimizer - 优化器
方法
  • train(dataloader, epochs) - 训练模型
示例
trainer = Trainer(model, criterion, optimizer) trainer.train(dataloader, epochs=100)

Evaluator

模型评估器，用于评估模型性能。

参数
  • model - 模型对象
  • criterion - 损失函数
方法
  • evaluate(dataloader) - 评估模型
示例
evaluator = Evaluator(model, criterion) accuracy = evaluator.evaluate(test_loader) print(f'Test Accuracy: {accuracy:.4f}')

utils 模块

utils 模块包含工具函数。

save_checkpoint

保存模型 checkpoint。

参数
  • state - 要保存的状态字典
  • filename - 保存文件名
示例
save_checkpoint({'model': model.state_dict(), 'optimizer': optimizer.state_dict()}, 'model.pth')

load_checkpoint

加载模型 checkpoint。

参数
  • filename - 加载文件名
返回值

保存的状态字典

示例
checkpoint = load_checkpoint('model.pth') model.load_state_dict(checkpoint['model']) optimizer.load_state_dict(checkpoint['optimizer'])

示例代码

LeNet 模型

以下是使用 EnNeuro 实现 LeNet 模型的示例：

from eneuro.base import Tensor, conv2d, linear, relu, softmax, pooling from eneuro.nn import Module, CrossEntropyLoss, SGD class LeNet(Module): def __init__(self): super().__init__() # 卷积层 self.conv1 = conv2d(1, 6, kernel_size=5, stride=1, padding=2) self.conv2 = conv2d(6, 16, kernel_size=5, stride=1, padding=0) # 全连接层 self.fc1 = linear(16 * 5 * 5, 120) self.fc2 = linear(120, 84) self.fc3 = linear(84, 10) def forward(self, x): # 第一层卷积 x = relu(self.conv1(x)) x = pooling(x, kernel_size=2, stride=2) # 第二层卷积 x = relu(self.conv2(x)) x = pooling(x, kernel_size=2, stride=2) # 展平 x = x.reshape(-1, 16 * 5 * 5) # 全连接层 x = relu(self.fc1(x)) x = relu(self.fc2(x)) x = softmax(self.fc3(x)) return x # 创建模型 model = LeNet()

模型训练与评估

from eneuro.data import Dataset, DataLoader from eneuro.train import Trainer, Evaluator # 准备数据 # ... # 创建数据加载器 train_loader = DataLoader(train_dataset, batch_size=64, shuffle=True) test_loader = DataLoader(test_dataset, batch_size=64, shuffle=False) # 创建模型、损失函数和优化器 model = LeNet() criterion = CrossEntropyLoss() optimizer = SGD(model.parameters(), lr=0.01, momentum=0.9) # 创建训练器 trainer = Trainer(model, criterion, optimizer) # 训练模型 trainer.train(train_loader, epochs=10) # 创建评估器 evaluator = Evaluator(model, criterion) # 评估模型 accuracy = evaluator.evaluate(test_loader) print(f'Test Accuracy: {accuracy:.4f}')

常见问题

Q: EnNeuro 与其他深度学习框架有什么区别？

A: EnNeuro 是一个轻量级的深度学习框架，专注于卷积神经网络的设计与实现。它提供了简洁的API接口，易于学习和使用，适合教学和研究用途。

Q: 如何保存和加载模型？

A: 可以使用 utils 模块中的 save_checkpoint 和 load_checkpoint 函数来保存和加载模型。

from eneuro.utils import save_checkpoint, load_checkpoint # 保存模型 save_checkpoint({'model': model, 'optimizer': optimizer}, 'model.pth') # 加载模型 checkpoint = load_checkpoint('model.pth') model.load_state_dict(checkpoint['model']) optimizer.load_state_dict(checkpoint['optimizer'])

Q: 如何使用 GPU 加速？

A: 目前 EnNeuro 主要基于 NumPy 实现，不直接支持 GPU 加速。未来版本可能会添加 GPU 支持。
